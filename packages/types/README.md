# Types Package

## Overview

This package provides TypeScript types and interfaces.

## Installation

### npm

```bash
npm install @ethereum-multicall/types
```

### yarn

```bash
yarn add @ethereum-multicall/types
```

### pnpm

```bash
pnpm add @ethereum-multicall/types
```

### bun

```bash
bun add @ethereum-multicall/types
```
