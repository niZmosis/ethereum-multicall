# Types Package

## Overview

This package provides TypeScript types and interfaces.

## Installation

### npm

```bash
npm install @multicall-toolkit/types
```

### yarn

```bash
yarn add @multicall-toolkit/types
```

### pnpm

```bash
pnpm add @multicall-toolkit/types
```

### bun

```bash
bun add @multicall-toolkit/types
```
