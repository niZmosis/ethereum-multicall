# Utils Package

## Overview

This package provides a set of utility functions.

## Installation

### npm

```bash
npm install @multicall-toolkit/utils
```

### yarn

```bash
yarn add @multicall-toolkit/utils
```

### pnpm

```bash
pnpm add @multicall-toolkit/utils
```

### bun

```bash
bun add @multicall-toolkit/utils
```
