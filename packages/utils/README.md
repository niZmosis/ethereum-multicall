# Utils Package

## Overview

This package provides a set of utility functions.

## Installation

### npm

```bash
npm install @ethereum-multicall/utils
```

### yarn

```bash
yarn add @ethereum-multicall/utils
```

### pnpm

```bash
pnpm add @ethereum-multicall/utils
```

### bun

```bash
bun add @ethereum-multicall/utils
```
